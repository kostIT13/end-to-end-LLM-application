Feature Store: централизованное управление признаками

Feature Store — это централизованное хранилище признаков (features) для машинного обучения. Решает проблему разрозненности признаков между обучением и инференсом. Основные компоненты: Online Store — низколатентное хранилище для real-time инференса (Redis, DynamoDB, Cassandra). Offline Store — аналитическое хранилище для обучения (S3, BigQuery, Snowflake). Feature Registry — каталог признаков с метаданными (тип, источник, владелец). Serving API — API для получения признаков в реальном времени. Point-in-time joins — корректное соединение признаков на момент времени для обучения. Популярные Feature Store: Feast (open-source, от Tecton) — самый популярный, интеграция с Redis, Datastore, S3, GCS. Tecton — коммерческая платформа от создателей Feast. SageMaker Feature Store (AWS) — интегрирован с SageMaker. Hopsworks — open-source Feature Store. Feature Store — ключевой компонент зрелой ML-инфраструктуры, особенно для real-time ML.
