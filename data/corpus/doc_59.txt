Apache Spark для больших данных в ML

Apache Spark — это мощный движок для распределенной обработки больших данных. Spark MLlib — библиотека машинного обучения, встроенная в Spark. Преимущества: обработка данных, которые не помещаются в память одной машины. Единый пайплайн: ETL + обучение + инференс. Поддержка Python (PySpark), Scala, Java, R. Компоненты MLlib: Feature engineering — VectorAssembler, StringIndexer, OneHotEncoder, StandardScaler, PCA. Algorithms — линейная регрессия, логистическая регрессия, деревья решений, Random Forest, GBT, K-Means, LDA. Pipelines — конвейеры, аналогичные scikit-learn. Tuning — CrossValidator, TrainValidationSplit. Spark Structured Streaming — для real-time ML. Spark NLP — библиотека от John Snow Labs для NLP на Spark. Ограничения: итеративные алгоритмы (нейронные сети) работают медленнее, чем в PyTorch/TensorFlow. Spark лучше подходит для feature engineering и классических ML-алгоритмов на больших данных. Альтернативы: Dask, Ray, Modin.
