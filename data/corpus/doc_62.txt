PostgreSQL и pgvector для ML

PostgreSQL — это мощная реляционная база данных, которая с расширением pgvector становится полноценным решением для ML. pgvector добавляет тип данных vector и индексы для поиска по сходству. Возможности: хранение векторов размерностью до 2000. Операторы расстояния: L2 (евклидово), IP (скалярное произведение), Cosine (косинусное). Индексы: IVFFlat (быстрое построение, менее точный), HNSW (медленнее построение, быстрее и точнее поиск). Гибридный поиск: комбинация векторного поиска с SQL-фильтрацией по метаданным. Преимущества: не нужна отдельная векторная БД, ACID транзакции, знакомый SQL, богатый экосистема PostgreSQL. Ограничения: pgvector — ANN, не точный поиск. Для очень больших датасетов (миллиарды векторов) лучше подходят специализированные векторные БД. pgvector — отличный выбор для старта и средних проектов, используется в production многими компаниями. Альтернативы: pg_embedding (Neon), Lantern.
