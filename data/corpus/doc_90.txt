ML в рекомендациях контента (YouTube, TikTok)

Рекомендации контента — ключевой компонент платформ как YouTube, TikTok, Netflix, Spotify. Архитектура: Two-stage: Candidate Generation — генерация кандидатов (сотни из миллионов). Ranking — ранжирование кандидатов. Candidate Generation: Collaborative Filtering, Content-based, Two-Tower Model (эмбеддинги пользователя и видео). YouTube: Deep Neural Network для candidate generation, затем deep ranking с cross features. TikTok: сильный упор на user engagement, RL для оптимизации долгосрочного удержания. Netflix: персонализированные ряды (rows), Contextual Bandits для A/B тестов. Spotify: комбинация коллаборативной и контентной фильтрации, аудио-анализ. Features: user features (история, демография), item features (метаданные, контент), context features (время, устройство). Objectives: CTR, Watch Time, Engagement, Retention, Satisfaction. Проблемы: filter bubble, echo chamber, холодный старт, свежесть контента. Рекомендации — двигатель пользовательского опыта.
