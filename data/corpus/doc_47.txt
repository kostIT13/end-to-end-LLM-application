MLflow: управление жизненным циклом ML

MLflow — это open-source платформа для управления жизненным циклом машинного обучения, разработанная Databricks. Компоненты: MLflow Tracking — логирование параметров, метрик, артефактов и моделей из экспериментов. API для Python, R, Java, REST. MLflow Projects — упаковка ML-кода в воспроизводимые запуски (conda, Docker). MLflow Models — стандартный формат для упаковки моделей с поддержкой множества фреймворков (scikit-learn, PyTorch, TensorFlow, ONNX). MLflow Model Registry — централизованное хранилище моделей с управлением версиями, стадиями (Staging, Production, Archived) и одобрениями. MLflow Deployments — API для деплоя моделей (SageMaker, Azure ML, Kubernetes). MLflow также поддерживает автоматическое логирование для популярных фреймворков. Интеграция с Databricks, Apache Spark. MLflow — стандарт де-факто для управления ML-экспериментами в индустрии.
