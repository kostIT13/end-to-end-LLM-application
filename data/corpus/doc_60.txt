Векторные базы данных для RAG

Векторные базы данных (Vector Databases) — это специализированные БД для хранения и поиска векторных эмбеддингов. Ключевая технология для RAG (Retrieval-Augmented Generation). Основные функции: хранение векторов большой размерности (768, 1024, 1536). Поиск по сходству (ANN — Approximate Nearest Neighbor). Гибридный поиск (векторный + keyword). Фильтрация по метаданным. Алгоритмы ANN: HNSW (Hierarchical Navigable Small World) — самый популярный, баланс скорости и точности. IVF (Inverted File Index) — быстрый, но менее точный. PQ (Product Quantization) — сжатие векторов. Популярные векторные БД: Pinecone — managed, прост в использовании. Weaviate — open-source, поддержка гибридного поиска. Qdrant — open-source, Rust-based, быстрый. Milvus — open-source, масштабируемый. Chroma — легковесная, для прототипирования. pgvector — расширение PostgreSQL, удобно если уже используете Postgres. FAISS (Facebook) — библиотека для ANN, не БД, но часто используется как движок.
