Ray: распределенные вычисления для ML

Ray — это open-source фреймворк для распределенных вычислений, разработанный в UC Berkeley. Ray упрощает параллельное и распределенное выполнение Python-кода. Основные компоненты: Ray Core — низкоуровневый API для распределенных задач и акторов. Ray Tune — распределенная оптимизация гиперпараметров. Ray Train — распределенное обучение моделей (PyTorch DDP, Horovod, DeepSpeed). Ray Serve — масштабируемый сервинг моделей с поддержкой A/B тестов, canary деплоя. Ray Data — распределенная обработка данных (замена Pandas на больших данных). Ray RLlib — библиотека для обучения с подкреплением. Интеграция с популярными фреймворками: PyTorch, TensorFlow, scikit-learn, XGBoost, LightGBM. Ray работает на кластерах (Kubernetes, AWS, GCP, Azure) и на одной машине. Преимущества: единый API для разных задач, отказоустойчивость, авто-масштабирование. Ray — стандарт для распределенного ML, альтернативы: Dask, Spark MLlib.
